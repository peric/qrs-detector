# QRS Detector
QRS complex detector that is able to process 24-hour ECG records of the "The Long-Term ST Database" (LTST DB)

## Dependencies

- Python 2.7.3
- Python libraries: pywfdb, matplotlib

## How to run - example

    python run_qrs_detector.py files/s20011
